Case Study

Blog Management Platform

A frontend-focused blog management platform with client dashboard, blog writing flow, draft saving, admin approval workflow, search/filter and responsive UI.

Project Goal

To build a practical frontend project that shows dashboard UI, form handling, dynamic rendering, localStorage data handling, and simple admin/client workflow.

Tech Used

  • HTML for page structure
  • CSS for dashboard layout and responsive UI
  • JavaScript for localStorage, blog actions, filters and dynamic content

What I Learned

  • Creating dashboard-style layouts
  • Handling forms and user actions with JavaScript
  • Using localStorage for frontend data persistence
  • Building a project that is simple but explainable in interviews
